== Description ==

Create awesome company page for Classified Maker to display information of companies, agency. 

### Classified Maker by http://pickplugins.com
* [Get Classified Maker!&raquo;](https://wordpress.org/plugins/classified-maker/)
* [Live Demo!&raquo;](http://www.pickplugins.com/demo/classified-maker/company/walmart/)
* [Plugin details!&raquo;](http://www.pickplugins.com/item/classified-maker)



<strong>Plugin Features</strong>

* Awesome company single page.
* User can follow companies.
* User can review star ratting to company.
* Related company on single company page.
* Company list by shortcode.
* Company submit page by shortcode.


<strong>Shortcodes</strong>

* New Company submit form:  `[classified_maker_company_submit]` [Company demo!&raquo;](http://www.pickplugins.com/demo/classified-maker/company-submit/)
* List of companies:  `[classified_maker_company_list]` [Company List!&raquo;](http://www.pickplugins.com/demo/classified-maker/company-list/)



<strong>Translation</strong>

Pluign is transaltion ready , please find the 'en.po' for default transaltion file under 'languages' folder and add your own translation. you can also contribute in translation, please contact us http://www.pickplugins.com/contact/


== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>Classified Maker</strong>" activate it.<br />
